### Control:
```
Go left: a
Go right: d
Jump: w or space
Shot: LMB
Gun change: 1, 2, 3
```
### Guns:
```
1: dmg: 50hp, reload: 1.0s
2: dmg: 30hp, reload: 0.7s
3: dmg: 10hp, reload: 0.2s
```
### Items:
```
- bomb: -30hp, spawn: 8s
- first aid kit: +50hp, spawn: 10s
```
